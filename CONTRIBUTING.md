# Contributing to roboat

Thank you for contributing to roboat. All contributions are welcome — bug reports, features, docs, and code.

## Getting Started

```bash
git clone https://github.com/Addi9000/roboat.git
cd roboat
pip install -e ".[all]"
pytest tests/ -v
```

## Making Changes

- Add type hints on all function signatures
- Add docstrings on every public method
- Return typed models, not raw dicts
- Run tests before submitting: `pytest tests/ -v`

## Submitting a Pull Request

1. Fork → branch → change → push
2. Open PR against `main`
3. Fill in the PR template

## Reporting Issues

Use the issue templates provided. Include version, Python version, OS, and a minimal reproduction.

## Questions

Open a Discussion on GitHub.
